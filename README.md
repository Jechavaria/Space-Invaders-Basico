# Space Invaders Basico
****************** Space Invaders Básico ******************

Un juego simple desarrollado en C# utilizando Windows Forms.

****************** Descripción ******************

Space Invaders Básico es una recreación sencilla del clásico juego de arcade. Permite a los jugadores controlar un tanque que debe defenderse de oleadas de enemigos espaciales. Incluye:

-> Movimiento del tanque a izquierda y derecha.
-> Disparo de proyectiles para eliminar enemigos.
-> Enemigos con patrones de movimiento y ataque.
-> Gestión de vidas y puntaje.
-> Detección de colisiones entre balas, enemigos y el tanque.
-> Interfaz gráfica basada en PictureBox para los elementos del juego.

****************** Características ******************

-> Control del Tanque: Se mueve con las teclas A/D o Izquierda/Derecha.
-> Disparo: Se activa con la barra espaciadora.
-> Oleadas de Enemigos: Se desplazan horizontalmente y bajan progresivamente.
-> Balas Enemigas: Algunos enemigos disparan hacia el tanque.
-> Sistema de Vidas: Se pierden al recibir impactos de balas enemigas o colisiones directas.
-> Game Over: Ocurre cuando el tanque pierde todas sus vidas o los enemigos alcanzan el suelo.

****************** Requisitos ******************

-> Visual Studio (2017 o superior recomendado).
-> .NET Framework (versión 4.7.2 o superior).
-> Conocimientos básicos de C# y Windows Forms.

****************** Uso ******************

Descarga o clona el repositorio.
Abre el proyecto en Visual Studio.
Compila la solución y ejecuta la aplicación.

Controles:

-> Teclas A/D o Izquierda/Derecha: Mueven el tanque.
-> Barra espaciadora: Dispara balas.

Condiciones de juego:

-> Los enemigos se desplazan y bajan gradualmente.
-> Al alcanzar el límite, los enemigos cambian de dirección.
-> Los enemigos pueden disparar balas aleatoriamente.
-> Si el tanque es alcanzado, pierde una vida.
-> Si los enemigos llegan al suelo o el tanque pierde todas las vidas, se muestra "Game Over".
